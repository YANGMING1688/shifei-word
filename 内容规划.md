
## 📝 写作流程

### 选题 → 大纲 → 写作 → 优化 → 发布

```
1. 选题（30 分钟）
   • 确定主题
   • 研究关键词
   • 分析竞品

2. 大纲（30 分钟）
   • 确定结构
   • 列出要点
   • 准备素材

3. 写作（1-3 小时）
   • 快速初稿
   • 不纠结细节
   • 完成比完美重要

4. 优化（30 分钟）
   • 修改错别字
   • 优化标题
   • 添加配图
   • SEO 优化

5. 发布（30 分钟）
   • 上传网站
   • 分发各平台
   • 回复评论
```
