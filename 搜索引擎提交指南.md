## 📤 提交搜索引擎（手动操作）

### 1. Google Search Console

**网址：** https://search.google.com/search-console

**步骤：**

1. **登录 Google 账号**
   - 使用 Gmail 账号登录

2. **添加属性**
   - 点击"添加属性"
   - 选择"域名"或"URL 前缀"
   - 输入：`shifei.world`

3. **验证所有权**（选择一种方式）
   
   **方式 A：DNS 验证（推荐）**
   ```
   1. 选择"DNS 验证"
   2. 复制 TXT 记录值
   3. 登录域名管理后台（万网/阿里云）
   4. 添加 TXT 记录
   5. 点击"验证"
   ```
   
   **方式 B：HTML 文件验证**
   ```
   1. 下载 Google 提供的 HTML 文件
   2. 上传到网站根目录
   3. 点击"验证"
   ```
   
   **方式 C：HTML 标签验证**
   ```
   1. 复制 meta 标签
   2. 粘贴到 index.html 的<head>中
   3. 提交到 GitHub
   4. 点击"验证"
   ```

4. **提交 Sitemap**
   ```
   1. 点击左侧"站点地图"
   2. 输入：sitemap.xml
   3. 点击"提交"
   ```

5. **请求索引**
   ```
   1. 在顶部搜索框输入网址
   2. 点击"请求编入索引"
   3. 等待审核（通常几小时）
   ```

## 🔧 常见问题

### Q1: 验证失败怎么办？
**A:** 检查 DNS 记录是否正确，等待 10-30 分钟再试

### Q2: Sitemap 提交后显示错误？
**A:** 检查 sitemap.xml 格式，确保 URL 可访问

### Q3: 多久能被索引？
**A:** Google 1-7 天，百度 3-14 天，Bing 1-7 天

### Q4: 如何加速索引？
**A:** 
- 每天更新内容
- 多平台分发（知乎、掘金等）
- 获取高质量外链
- 使用主动推送 API
