# 搜索引擎验证文件

## Google Search Console 验证

**验证方式：** HTML 标签验证

**步骤：**
1. 访问：https://search.google.com/search-console
2. 登录 Google 账号
3. 点击 "添加属性"
4. 输入：`shifei.world`
5. 选择 "HTML 标签" 验证方式
6. 复制 meta 标签（类似：`<meta name="google-site-verification" content="xxxxxxxxxx">`）
7. **将 meta 标签内容告诉我**

**我会：**
- 添加到 index.html 的<head>中
- 提交到 GitHub
- 你返回 Google 点击 "验证"
