## 🛒 步骤 1：注册阿里云账号

### 1.1 注册账号

```
1. 访问 www.aliyun.com
2. 点击右上角 "免费注册"
3. 选择"个人账号"
4. 输入手机号，获取验证码
5. 设置密码
6. 完成注册
```

### 1.2 实名认证（重要！）

```
1. 登录后访问：account.aliyun.com
2. 点击 "实名认证"
3. 选择"个人实名认证"
4. 上传身份证正反面
5. 完成人脸识别
6. 等待审核（通常 10-30 分钟）
```

### 1.3 购买域名 shifei.ai

```
1. 访问：wanwang.aliyun.com
2. 搜索框输入：shifei.ai
3. 点击"查询"
4. 如果可用，点击"加入购物车"
5. 结算付款（支持支付宝/微信）
6. 完成购买
```

### 1.4 域名实名认证

```
1. 访问：domain.aliyun.com
2. 找到 shifei.ai 域名
3. 点击"实名信息认证"
4. 上传身份证照片
5. 填写信息（与阿里云账号一致）
6. 提交审核（1-3 天）
```

**⚠️ 注意：域名实名认证必须与备案主体一致！**

## 📱 步骤 3：提交 ICP 备案申请

### 3.1 进入备案系统

```
1. 访问：beian.aliyun.com
2. 用阿里云账号登录
3. 点击"开始备案"
```

### 3.2 填写备案信息

#### 第一步：验证主办单位信息
```
• 主办单位性质：个人
• 主办单位名称：你的姓名（自动填充）
• 证件类型：身份证
• 证件号码：你的身份证号
• 真实性核验：人脸识别
```

#### 第二步：填写网站信息
```
• 网站名称：世飞创业心得
• 域名：shifei.ai
• 网站负责人：你的姓名
• 负责人手机号：你的实名手机号
• 负责人邮箱：你的邮箱
• 网站内容：个人技术博客，分享一人公司创业经验
```

#### 第三步：上传材料
```
• 身份证正反面照片
• 个人照片（人脸识别时拍摄）
• 域名证书（系统自动获取）
• 备案真实性核验单（系统生成，需打印签字后上传）
```

#### 第四步：选择接入方式
```
• 选择"阿里云"作为接入商
• 选择你购买的服务器实例
```

### 3.3 提交审核

```
1. 检查所有信息是否正确
2. 阅读并同意备案服务协议
3. 点击"提交审核"
```

## 📲 步骤 4：短信核验

### 4.1 阿里云初审

```
• 阿里云工作人员审核材料（1 个工作日内）
• 如有问题会电话联系你
• 审核通过后提交到工信部
```

### 4.2 工信部短信核验

```
1. 工信部发送短信到你的手机
2. 短信包含验证码和核验网址
3. 访问工信部网站：beian.miit.gov.cn
4. 输入验证码完成核验
5. 核验通过后进入管局终审
```

**⚠️ 重要：收到短信后 24 小时内完成核验！**

---

## ⏳ 步骤 5：等待管局终审

### 5.1 审核时间

```
• 不同省份管局时间不同
• 通常：7-20 个工作日
• 广东、浙江较快（7-10 天）
• 北京、上海较慢（15-20 天）
```

### 5.2 查询进度

```
1. 访问：beian.aliyun.com
2. 登录后查看备案进度
3. 或访问：beian.miit.gov.cn 查询
```

### 5.3 审核结果

```
✅ 通过：获得 ICP 备案号（如：京 ICP 备 12345678 号）
❌ 驳回：根据驳回原因修改后重新提交
```

### 6.2 域名解析到国内服务器

```
1. 访问阿里云域名控制台
2. 找到 shifei.ai 域名
3. 点击"解析"
4. 添加 A 记录：
   • 主机记录：@
   • 记录值：你的服务器 IP
   • TTL：10 分钟
5. 添加 CNAME 记录：
   • 主机记录：www
   • 记录值：你的服务器 IP 或域名
```

### 6.3 网站迁移（从 Vercel 到阿里云）

```
1. 将网站文件上传到阿里云服务器
2. 配置 Nginx/Apache
3. 测试访问
4. 确认正常后，Vercel 可保留或关闭
```

## ⚠️ 常见问题

### Q1: 备案期间网站能访问吗？
```
A: 可以！
   • 备案期间网站可以继续放在 Vercel
   • 用 Vercel 的临时域名访问
   • 备案通过后再迁移到国内
```

### Q2: 个人备案能做什么类型的网站？
```
A: 个人备案适合：
   ✅ 个人博客
   ✅ 技术笔记
   ✅ 创业日记
   ✅ 作品集
   
   ❌ 不适合：
   • 电商网站
   • 论坛社区
   • 新闻资讯
   • 视频网站
```

### Q3: 备案被驳回怎么办？
```
A: 根据驳回原因修改：
   • 网站名称不合适 → 改名
   • 材料不清晰 → 重新上传
   • 信息不一致 → 修改一致
   • 网站内容违规 → 调整内容
```

### Q4: 备案成功后有什么要求？
```
A: 
   • 网站底部必须显示 ICP 备案号
   • 备案号链接到工信部网站
   • 网站内容要与备案一致
   • 不要做违法违规内容
   • 服务器到期要及时续费或注销备案
```

### Q5: 可以找人代办备案吗？
```
A: 不建议！
   • 备案需要本人人脸识别
   • 需要本人手机短信核验
   • 信息必须真实
   • 代办有风险
```
