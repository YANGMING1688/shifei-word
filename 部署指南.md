## 📋 部署步骤（按顺序执行）

### 步骤 1：创建 GitHub 账号（如果没有）
```
访问：github.com
注册免费账号
```

### 步骤 2：创建仓库并上传代码
```bash
# 在终端执行以下命令

cd /Users/yangming/.openclaw-openclaw/workspace/shifei-opc-blog

# 初始化 git
git init

# 添加所有文件
git add .

# 提交
git commit -m "初始版本 - 世飞 opc 一人公司创业心得"

# 在 GitHub 创建新仓库后，执行：
git remote add origin https://github.com/你的用户名/shifei-opc-blog.git
git branch -M main
git push -u origin main
```

### 步骤 3：部署到 Vercel
```
1. 访问 vercel.com
2. 点击 "Sign Up" 用 GitHub 登录
3. 点击 "Add New Project"
4. 选择 "Import Git Repository"
5. 找到 shifei-opc-blog 仓库
6. 点击 "Import"
7. 保持默认设置，点击 "Deploy"
8. 等待部署完成（约 1 分钟）
9. 获得免费网址：shifei-opc-xxx.vercel.app
```

### 步骤 4：购买域名
```
推荐平台：
• namesilo.com（便宜，约$8/年）
• cloudflare.com（管理方便）
• 阿里云 wanwang.aliyun.com（国内）

推荐域名：
• shifeiopc.com
• shifei-creator.com
• shifei-daily.com
• 世飞创业.com（中文域名）
```

### 步骤 5：绑定域名到 Vercel
```
1. 登录 Vercel 控制台
2. 选择你的项目
3. 点击 "Settings" → "Domains"
4. 输入你买的域名（如 shifeiopc.com）
5. 点击 "Add"
6. 按提示设置 DNS：
   
   类型    名称    值
   A       @       76.76.21.21
   CNAME   www     cname.vercel-dns.com

7. 等待 DNS 生效（5 分钟 - 24 小时）
```

### 步骤 6：提交到搜索引擎（关键！）

#### Google 搜索
```
1. 访问 search.google.com/search-console
2. 添加你的网站
3. 验证所有权（用 Vercel 的 DNS 验证）
4. 提交 sitemap.xml
5. 使用 "URL 检查" 工具提交首页
6. 等待收录（通常 3-7 天）
```

#### 百度搜索
```
1. 访问 ziyan.baidu.com
2. 注册并添加网站
3. 验证所有权
4. 提交 sitemap.xml
5. 使用 "快速收录" 功能
6. 等待收录（通常 1-4 周）
```

#### Bing 搜索
```
1. 访问 www.bing.com/webmasters
2. 添加你的网站
3. 验证所有权
4. 提交 sitemap.xml
5. 等待收录（通常 3-7 天）
```

## 📈 加速收录的技巧

### 1. 持续更新内容
```
• 每周至少发布 1-2 篇新文章
• 搜索引擎喜欢活跃的网站
• 固定时间更新更好（如每周一、四）
```

### 2. 外部链接建设
```
• 在社交媒体分享你的文章
• 在知乎、小红书等平台提及你的网站
• 和其他博主交换友链
• 在论坛签名档添加网站链接
```

### 3. 内容 SEO 优化
```
• 文章标题包含关键词
• 每篇文章 800 字以上
• 使用小标题分段
• 添加图片和描述
```

### 4. 社交媒体推广
```
• 微信公众号：发布文章并附上网站链接
• 知乎：回答问题时引用你的内容
• 小红书：分享创业心得 + 网站链接
• Twitter/X：发布创业进展
```
