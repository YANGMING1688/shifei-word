## 📋 Token 获取步骤（2 分钟）

### 第 1 步：访问百度资源平台
https://ziyuan.baidu.com/site/index#/

### 第 2 步：登录并验证网站
1. 使用百度账号登录
2. 添加网站：`shifei.world`
3. 验证所有权（HTML 标签验证最快）

### 第 3 步：获取 token
1. 点击 **"搜索服务"** → **"普通收录"**
2. 切换到 **"资源提交"** → **"API 提交"**
3. 复制 `token=` 后面的字符串
