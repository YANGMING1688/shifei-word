## 📋 配置清单

### 第 1 步：GitHub 认证配置（必做）

#### 方式一：Personal Access Token（推荐）

```bash
# 1. 生成 Token
# 访问：https://github.com/settings/tokens
# 权限：repo (Full control)

# 2. 配置到 macOS 密钥链
git credential-osxkeychain erase
host=github.com
protocol=https
^D  # 按 Ctrl+D

git credential-osxkeychain store
host=github.com
protocol=https
username=YANGMING1688
password=你的_TOKEN
^D  # 按 Ctrl+D

# 3. 测试
cd /Users/yangming/.openclaw-openclaw/workspace/shifei-opc-blog
git push origin main
```

#### 方式二：SSH 密钥

```bash
# 1. 生成密钥
ssh-keygen -t ed25519 -C "yangming@opctool.com"
# 一路回车

# 2. 添加公钥到 GitHub
cat ~/.ssh/id_ed25519.pub
# 复制内容到：https://github.com/settings/keys

# 3. 切换为 SSH
cd /Users/yangming/.openclaw-openclaw/workspace/shifei-opc-blog
git remote set-url origin git@github.com:YANGMING1688/shifei-word.git
git push origin main
```
